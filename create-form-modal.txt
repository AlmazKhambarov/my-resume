CreateForm — это универсальный компонент для отображения модального окна с формой. 
Он поддерживает динамически определяемые поля, возможность разделения формы на две части и добавления пользовательских элементов. 
Компонент также обрабатывает проверки на изменения формы, предупреждая пользователя о несохранённых данных.

Основные пропсы:

fields — массив конфигураций полей формы. Ожидает поля, которые определены в компоненте FormFieldComponent.
initialValues — начальные значения для заполнения формы.
isSplit — разделяет форму на две части, если установлено в true.
handleSubmit — функция, которая вызывается при подтверждении формы, принимает текущие значения формы.
element — произвольный элемент, который можно добавить к форме (например, дополнительный блок информации).
onFormChange — коллбэк, вызываемый при изменении значений формы.
open и setOpen — управление видимостью модального окна.
headerText — текст заголовка модального окна.
modalSize — размер модального окна, по умолчанию "lg".
Основные функции:

handleFormSubmit — вызывает handleSubmit, а затем сбрасывает форму.
onCloseModal — проверяет, есть ли изменения, и если они есть, вызывает предупреждающее окно.
renderFields — рендерит поля формы на основе массива fields.
